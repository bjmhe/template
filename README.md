A template for creating a TypeScript package.

## Install

<!-- automd:pm-install dev auto=false -->

```sh
# npm
npm install -D template

# yarn
yarn add -D template

# pnpm
pnpm add -D template

# bun
bun install -D template

# deno
deno install --dev npm:template

# vp
vp install -D template
```
